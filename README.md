# Packet-Tracer-Python-Viewer
Herramienta que lee archivos de topología de Cisco Packet Tracer (.xml, descifrados previamente) y extrae información de los dispositivos de red. Actualmente calcula la tabla de ruteo (show ip route) de cada router, con soporte planeado para switches y otros dispositivos. Basado en el trabajo de descifrado de Punkcake21/Unpacket.
